Welcome to Clamp

Clamp is a Command Line Apache MySQL PHP for Mac OS X. It aims at being ultra simple to use and configure.

  • Uses internal Apache.
  • All data are self contained.
  • Configuration using a JSON file.

Installation

$ curl http://jide.github.io/clamp/install.sh | sh

Quickstart

To quickly get your servers running, go to your project folder and type : 

$ clamp

Use Ctrl + C to exit.

Everything is self-contained in a .clamp folder inside your project: The database files, the logs and the socket and PID files.
